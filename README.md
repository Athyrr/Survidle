# Survidle

# Requirements
- CMake ≥ 3.10
- A compilo C++20 (MSVC, Clang, GCC, Ninja ..)
- vcpkg (Optionnal) 

#  Build (CLion, Visual Studio)
It will generate a build folder 

- CLion : open the project folder.
- VS : File > Open > Folder + `Survidle/`
